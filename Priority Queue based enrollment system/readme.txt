README - Siddharth Chhatbar

This program creates an enrollment table for a class with maximum 
capacity equal to 50.
The enrollment table is implemented using a linked list.
There is also a Priority Queue on the basis of which students will be
enrolled using the priority of their falulty.
Falulty(number) in decreasing order of their priority:
SCI(4), ENG(3), BUS(2), ART(1), EDU(0) 
The user can add or drop students by providing their information in a text file.
